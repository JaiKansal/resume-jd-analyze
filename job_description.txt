About Us:
We are a fast-growing tech startup building AI-assisted platforms for hiring, logistics, and ed-tech. Our mission is to simplify digital workflows using the latest in AI, automation, and web technologies. We're looking for a sharp, motivated Junior Full Stack Developer to join our remote team and contribute to next-gen web products from day one.

Role Overview:
As a Junior Developer, you’ll help design and build robust, scalable, and responsive web applications using modern technologies like React, TypeScript, and Next.js. You’ll also work on internal developer tools powered by AI copilots to enhance efficiency across our engineering team. You should be comfortable writing clean code, iterating quickly, and solving algorithmic challenges when needed.

- Occasionally contribute to backend APIs and microservices
- Help with algorithmically driven features, such as resume ranking, skill matching, or user recommendations

Requirements:
- Experience with JavaScript, React, TypeScript, and modern web development practices
- Familiarity with Next.js or other full-stack React frameworks
- Basic understanding of data structures, algorithms, and code optimization
- Ability to build mobile-first responsive interfaces
- Knowledge of version control tools (e.g., Git/GitHub)
- Comfortable using AI tools like ChatGPT, GitHub Copilot, or Cursor

Bonus Skills (Nice to Have):

- Prior experience with problem setting, contests, or coding platforms
